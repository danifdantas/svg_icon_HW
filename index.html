Web Page Icon Set

This is the Play Icon. Whenerver there is a video this icon is used to show the user where she/he should press to play the video.

This is the Pause Icon. Whenerver there is a video this icon is used to show the user where she/he should press to pause the video.

This is the Microphone Icon. Whenerver there is audio to be played, this icon is used to show the user where she/he should press to play the audio, or control the volume.

This is the Headphone Icon. Whenerver there is audio to be played, this icon is used to show the user where she/he should press to connect with a bluetooth device to hear the audio through headphones.

This is the Fast Shopping Cart Icon. Whenerver there is shopping in a website, this icon is used to show the user where she/he should press to send items to the cart, or make a fast purchase.

This is the Search Icon. Whenerver the user wants to search content in the page, this icon shows the user where to make a search.

This is the Calendar Icon. It shows the user where she/he should press to see the calendar, or add activities in their calendar.

This is the Mail Icon. It shows the user where to find contact information.

This is the Home Icon. It indicates to the user where the home page is.

This is the Location Icon. It either indicated the location of the user, or the location she/he is looking for.
